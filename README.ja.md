# **Music Macro Language Compiler**

[ENGLISH](/README.md)

## Description

MML を 標準MIDIファイル へコンパイル(変換)するクラスライブラリです。<br>
標準MIDIファイル から MML への変換もサポートしています。
- C++20 での実装です。
- コマンドラインアプリとしてビルドできる形になっています。
  - mmltosmf : MML → 標準MIDIファイル
  - smftomml : 標準MIDIファイル → MML
- MML の文法には決まった規格があるわけではありませんが、暗黙の標準のようなものは踏襲するようにしています。

## Demo Page

[https://rlib-mml.thinkridge.jp/](https://rlib-mml.thinkridge.jp/)

MML の作成から再生までをブラウザ上で行える形にまとめています。

C++20 環境でコンパイルできます。ビルドには boost が必要です。

|r[音長]|  休符です。直後に音長を指定できます(省略可)。 | // 16分休符です。<br>r16|
|^[音長]|  タイです。直後に音長を指定できます(省略可)。<br>直前の音符の音長を足します。<br>直前が音符ではない場合は休符と同じです。| // 4分音符３つ分の長さのドです。<br>c4^4^4<br><br>// 4分音符3つ分の長さのドです。途中でテンポを変えています。<br>// 音符と ^ の間に別の命令を記述することが可能です。<br>c4 t50 ^4 t20 ^4
|l[音長]|  デフォルト音長です。<br>音符及び休符で音長指定を省略した場合に採用する値です| // 8分音符のドレミです<br>l8 cde|
|<<br>> | オクターブアップ(<) ダウン(>) です。<br>1オクターブの範囲は ド(c)～シ(b) の12音です。| // ド と １オクターブ上のド を２回繰り返します<br>c\<c>c\<c |

### 音長
- 音の長さ(ここでは音長と表記します)の指定方法です。

| 表記  |  説明  |例|
| ---- | ---- | ---- |
|1～192<br>音符表記| 〇分音符の値を指定します。<br>12分音符や24分音符という表記も可です| // 4分音符のドです<br>c4|
|!1～!99999<br>ステップ数表記| ステップ数で指定します。4分音符が480です。| // 8分音符のドです<br>c!240|
|.<br>付点表記| 付点〇分音符というところの付点を意味します。<br>複数記述することで複付点音符の意味(直前の音長の半分を足す)になります。| // 付点4分音符のドです<br>c4.<br><br>// 4分音符と8分音符と16分音符と32分音符を足した音長のドです<br>c4...|
|+ - 数値<br>音長の加減算指定|音長値を加減算します。|// 8分音符と4分音符を足した音長のドです<br>c8+4<br><br> // 32分音符のドレ、全音符から32分音符2つ分を短くしたミです。<br>c32d32e1-32-32<br><br>// この場合は、4分音符のレ♭です。ご注意下さい<br>d-4|

## 文字列

文字列を指定する箇所では、以下の書式が使えます。

|  表記  |  説明  |例|
| ---- | ---- | ---- |
| ○○○ | 頭文字が英字で、かつ英数字のみ使われている文字列はこの指定が可能です。| trumpet |
| "○○○" | " から " までが文字列です。空白や記号を使う場合にも使えます。　| "drum part"
| R"\*\*(○○○)\*\*" | 空白や記号や " を使う場合にも使えます。<br>シーケンスの中にシーケンスを定義するような場合は、** を一意の文字列にすることで、文字列定義の中にさらに文字列定義があるようなケースに対応できます　| R"(drum)"<br><br> R"ch1( mml:R"(drum)" )ch1"

## 相対指定

相対指定が可能な箇所では、現在値を基準に値を指定できます。
複合演算子で四則演算をサポートします。少数表記も可能です。

|  表記  |  説明  | 例 |
| ---- | ---- | ---- |
| +=n | 現在値にnを足します| // ベロシティを+10しています<br> v+=10 cde |
| -=n | 現在値からnを引きます| // エクスプレッション100 → 96 (95.6を四捨五入) → 95(95.2を四捨五入)<br>Ep(100) c Ep(-=4.4) d Ep(-=0.4) e|
| \*=n | 現在値にnを掛けます| // ボリューム100 → 127 (130が範囲内にクリップされます) →  117 (現在値130に0.9を掛けます)<br>V(100) c V(\*=1.3) d V(\*=0.9) e|
| \/=n | 現在値をnで割ります| // ボリューム100 → 50 → 25<br>V(100) c V(/=2) d V(/=2) e|
